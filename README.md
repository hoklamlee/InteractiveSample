# InteractiveSample

## Prerequisites
1. Install VLC in the Correct Location
Make sure VLC is installed in the correct directory. The default installation path for VLC is:

	`C:\Program Files\VideoLAN\VLC\vlc.exe`

	If VLC is installed in a different location, be sure to update the path in your code accordingly.

2. Enable remote control in VLC
	1. Open VLC.
	2. Go to Tools ¡÷ Preferences.
	3. Select Show settings ¡÷ All (bottom left).
	4. Navigate to Interface ¡÷ Main interfaces and check "Web"
	5. Restart VLC.
